# website-responsivo-brownie-barkery
 Criei um website para uma doceria, o qual atende o cliente no formato desktop como no formato para dispositivos móveis, totalmente responsivo.
 
## Qual a expectativa do projeto?

O objetivo foi criar um website elegante, atraente e convidativo, para que os usuários possam sentir-se atraídos pelos produtos fornecidos pela doceria Barkey. 
<br>
Com isso criamos uma estratégia para deixar os usuários confortáveis e tentados a conhecer todos os produtos do menu.

## Foi desenvolvido com

Este foi desenvolvido com a linguagem de marcação de hipertexto HTML, foi usado o CSS para estilizar e fazer algumas animações com o animate, também foi usado a linguagem de programação JavaScript usando recursos do JQuery. Foi usado a IDE Visual Studio Code.
